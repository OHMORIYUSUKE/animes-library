## ローカルで動かす

```sh
$ git clone git@github.com:OHMORIYUSUKE/animes-library.git
$ cd animes-library
$ npm install
$ npm run generate
$ npm run migrate
$ cd src/app/server/app
$ npm install
$ npm run seed
$ cd -
$ npm run dev
```

`localhost:3000`にアクセス
